How to create links in html

Introduction

Before reviewing how to link information within an html file, it is important to note that there are different categories of links. External and internal.

External links attach information to the html file from an external source, such as from another webpage.

Internal links attach files from within a local file directory, or to another part of the same html document.

However, whether or not we are using an internal or an external link, the link is always wrapped within an anchor element <a>. This is then followed by a hyperlink reference 'href', an equals sign '=', a link wrapped in quotations, and the displayed name of the link:


<a href="">name</a>

External linking

Here is an example of an anchor element linking to another webpage. This is reffered to as an absolute path.


<a href="https://www.w3schools.com/html/html_filepaths.asp" target=_blank>W3schools - HTML File Paths</a>

Additionally there is a target attribute included that tells a link how to open. In this case "_blank" tells the browser to open in a new window, or in the case of a modern browser, in a new tab.

Internal linking

Projects are often created in one main folder with different sections or subfolders within. The browser is aware of these sibling folders and can access their information via the use of relative paths. For example:


<a href="./contact.html">Contact</a>

the ./ tells the browser to look for a file within the current project folder. Note if you need to reach a file within a parent directory then use ../, this can be repeated as many times as necessary.

Internal linking (same page)

Websites can often become quite large and it can be difficult to quickly locate information. Thankfully navagation links can help the viewer get to where they need to be. These links can take the viewer to a new page (external link) or may jump to another section of the same webpage as shown in the example below:


  • <ol>
    1. <a href="#top">Top</a>
    2. <a href="#bottom">Bottom</a>
  • </ol>

In the case of same page internal linking the targeted element must have an id, and the target link is a string containing both the # character and the target elements id. Here are examples of the targeted elements:


  • <p id="top">This is the top of the page!</p>
  • <p id="bottom">This is the bottom of the page!</p>

Summary of the above information


Same page links must be attached to elements with an id!
Types of html links
External Internal internal (same page)
<a href="">name</a> <a href="./">name</a> <a href="#">name</a>
